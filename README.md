# DocMan 一款国产的Docker云开源平台

![仪表盘](http://ww2.sinaimg.cn/large/005Bpb8ijw1f9wb0ueh2ej31kw0vsn15.jpg)

URL：[DocMan](http://www.docman.org)

## 特色

- 图表统计
- 响应式
- 资产管理

## 功能
- 主机管理
- 应用管理
- 镜像仓库
- 监控报警
- 操作日记
- 云端意见反馈

## 安装

> `python2.7` + `Flask` + `Mysql`
> 
> 缺什么库安装什么库，推荐 pip 安装
